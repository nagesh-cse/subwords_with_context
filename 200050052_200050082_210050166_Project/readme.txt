To generate tokens run from 'code' folder directory:-
	`streamlit run generate_tokens.py <en or hi or mr>`
	1. Write 'en' for english model
	2. Write 'hi' for hindi model
	3. Write 'mr' for marathi model

1. 'english' , 'hindi' and 'marathi' folder contains the pretrained models 
2. Other results contains other experimentals results and plots for each language
3. Complete code for paper is at `https://github.com/MeLeLBGU/SaGe`
